#Calculating a tip
READ receipt total
SET tip to zero
IF service was good
  tip equals receipt total times twenty percent
ELSIF service was poor
  tip equals receipt total times ten percent
ENDIF
RETURN tip

#Weight-lifting
FOR each exercise in workout
  SET count to zero
  WHILE count is less than 4
    SET repetitions to zero
    WHILE repetitions is less than 11
      PERFORM given exercise
      INCREMENT repetition by one
    ENDWHILE
    INCREMENT count by one
  ENDWHILE
ENDFOR
